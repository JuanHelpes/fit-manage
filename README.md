# Fit Manage 💪📊

**Fit Manage** é uma aplicação web fullstack desenvolvida para o gerenciamento de fichas de treino entre **alunos** e **instrutores**, centralizando acompanhamento de treinos, histórico e evolução física em um único sistema.

O projeto foi idealizado para resolver problemas comuns em academias e treinamentos personalizados, como controle de fichas ativas, acompanhamento de progresso e comunicação eficiente entre aluno e instrutor.

> 🚧 **Projeto em desenvolvimento** — novas funcionalidades estão sendo implementadas continuamente.

---

## 🚀 Funcionalidades

### 👤 Aluno
Os alunos têm acesso a uma interface focada na execução e acompanhamento do treino:

- Visualização da **ficha de treino atual**
- Acompanhamento do treino em tempo real
- Edição da **carga de cada exercício**
- Controle e marcação de **tempo de descanso**
- Acesso ao **histórico de fichas**
- Registro de dados físicos:
  - Peso
  - Altura
  - Percentual de gordura
  - Medidas corporais (braço, cintura, perna)
- Tela de **evolução física**, com comparação mensal dos dados registrados

---

### 🧑‍🏫 Instrutor
O instrutor possui um painel administrativo completo para gestão dos alunos e fichas:

- **Dashboard** com indicadores:
  - Alunos ativos
  - Fichas ativas
  - Fichas próximas do vencimento
- Lista completa de alunos
- Sistema de **alertas**, incluindo:
  - Fichas expirando
  - Fichas expiradas há X dias
  - Fichas que vencem na semana
- Criação e gerenciamento de fichas de treino:
  - Seleção do aluno
  - Definição de objetivo
  - Data de início e fim
  - Montagem completa da ficha
- Disponibilização automática da ficha para o aluno
